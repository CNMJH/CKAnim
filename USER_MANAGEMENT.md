# 👥 用户管理功能使用指南

## 功能概述

管理员后台现已支持完整的用户管理功能，包括：
- ✅ 用户列表查看（支持搜索和筛选）
- ✅ VIP 等级设置（普通用户/VIP 月卡/VIP 年卡/永久 SVIP）
- ✅ VIP 过期时间管理
- ✅ 用户角色管理（普通用户/内容管理员/系统管理员）
- ✅ 密码重置
- ✅ 用户删除（仅系统管理员）

## 访问方式

1. 登录管理员后台：`https://admin.anick.cn`
2. 点击左侧菜单 **"👥 用户管理"**
3. 仅 `system_admin` 和 `content_admin` 角色可访问

## 功能详解

### 1️⃣ 用户列表

**显示信息：**
- ID、用户名、头像、邮箱
- 角色（普通用户/内容管理员/系统管理员）
- VIP 等级（普通用户/VIP 月卡/VIP 年卡/永久 SVIP）
- VIP 过期时间
- 收藏数量、个人参考库视频数量
- 注册时间

**搜索和筛选：**
- 搜索框：支持用户名或邮箱模糊搜索
- VIP 等级筛选：按 VIP 等级过滤用户

**分页：**
- 每页显示 20 条记录
- 支持上一页/下一页导航

### 2️⃣ 设置用户 VIP

**步骤：**
1. 点击用户行右侧的 **"编辑"** 按钮
2. 在弹窗中找到 **"VIP 设置"** 区域
3. 选择 VIP 等级：
   - **普通用户**：无上传权限
   - **VIP 月卡**：单文件 10MB，总空间 100MB
   - **VIP 年卡**：单文件 100MB，总空间 10GB
   - **永久 SVIP**：单文件 200MB，总空间 50GB
4. 设置过期时间（可选）：
   - 留空则自动设置（月卡 30 天，年卡 365 天）
   - 永久 SVIP 无需设置过期时间
5. 点击 **"💾 保存 VIP 设置"**

**示例：给用户 qwer111 设置 VIP 月卡**
```
1. 找到用户 qwer111
2. 点击"编辑"
3. VIP 等级选择"VIP 月卡"
4. 过期时间留空（自动设置为 30 天后）
5. 点击"保存 VIP 设置"
```

### 3️⃣ 管理用户角色

**仅系统管理员可操作**

**步骤：**
1. 点击用户行右侧的 **"编辑"** 按钮
2. 在弹窗中找到 **"角色设置"** 区域
3. 选择用户角色：
   - **普通用户**：仅可使用前台功能
   - **内容管理员**：可管理内容（游戏、分类、角色、动作等）
   - **系统管理员**：完整权限（包括用户管理、数据库管理等）
4. 点击 **"🔐 保存角色"**

### 4️⃣ 重置用户密码

**步骤：**
1. 点击用户行右侧的 **"编辑"** 按钮
2. 在弹窗中找到 **"账户安全"** 区域
3. 点击 **"🔑 重置密码"**
4. 输入新密码（至少 6 位）
5. 点击 **"确认重置"**

**注意：** 重置后需将新密码告知用户

### 5️⃣ 删除用户

**仅系统管理员可操作**

**步骤：**
1. 点击用户行右侧的 **"删除"** 按钮
2. 确认删除提示
3. 用户将被永久删除

**限制：**
- 不能删除管理员账号（系统管理员/内容管理员）
- 删除后数据不可恢复

## API 接口

### 获取用户列表
```bash
GET /api/admin/users?page=1&limit=20&search=&vipLevel=all

# 参数说明
page: 页码（默认 1）
limit: 每页数量（默认 20）
search: 搜索关键词（用户名或邮箱）
vipLevel: VIP 等级筛选（all/none/vip_monthly/vip_yearly/vip_lifetime）
```

### 获取用户详情
```bash
GET /api/admin/users/:id
```

### 更新用户 VIP
```bash
PUT /api/admin/users/:id/vip
Content-Type: application/json
Authorization: Bearer <TOKEN>

{
  "vipLevel": "vip_monthly",  // none | vip_monthly | vip_yearly | vip_lifetime
  "vipExpires": "2026-04-27T20:40:00Z"  // 可选，留空则自动计算
}
```

### 重置用户密码
```bash
PUT /api/admin/users/:id/reset-password
Content-Type: application/json
Authorization: Bearer <TOKEN>

{
  "newPassword": "NewPassword123"
}
```

### 更新用户角色
```bash
PUT /api/admin/users/:id/role
Content-Type: application/json
Authorization: Bearer <TOKEN>

{
  "role": "content_admin"  // user | content_admin | system_admin
}
```

### 删除用户
```bash
DELETE /api/admin/users/:id
Authorization: Bearer <TOKEN>
```

## VIP 等级说明

| VIP 等级 | 代码 | 单文件限制 | 总空间限制 | 说明 |
|---------|------|-----------|-----------|------|
| 普通用户 | `none` | 0 MB | 0 MB | 不支持上传 |
| VIP 月卡 | `vip_monthly` | 10 MB | 100 MB | 有效期 30 天 |
| VIP 年卡 | `vip_yearly` | 100 MB | 10 GB | 有效期 365 天 |
| 永久 SVIP | `vip_lifetime` | 200 MB | 50 GB | 永久有效 |

**注意：** VIP 限制可在"个人参考库"页面统一配置

## 常见问题

### Q: 如何批量设置 VIP？
A: 目前支持单个用户设置，如需批量操作可联系开发团队。

### Q: VIP 过期后会自动降级吗？
A: 不会自动降级，但上传时会检查过期时间，过期后无法上传。

### Q: 如何查看用户的上传记录？
A: 在用户编辑弹窗中可查看个人参考库视频数量，详细记录功能开发中。

### Q: 删除用户会删除其上传的视频吗？
A: 不会，删除用户时不会级联删除其上传的内容。
