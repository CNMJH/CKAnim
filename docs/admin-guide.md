# CKAnim 管理员后台使用指南

## 快速开始

### 第一次使用时的完整流程

```
1. 创建动作（全局）
   ↓
2. 创建分类（用于角色分类）
   ↓
3. 创建角色
   ↓
4. 上传视频（绑定角色 + 动作）
```

---

## 详细步骤

### 1️⃣ 创建动作（全局）

**访问**: http://localhost:3003/actions

**操作**:
1. 点击 **"➕ 新建动作"**
2. 填写信息：
   - **名称**: 攻击
   - **代码**: attack（英文标识符，用于程序调用）
   - **描述**: 普通攻击动作（可选）
   - **排序**: 0（数字越小越靠前）
   - **发布**: ✅ 勾选
3. 点击 **"保存"**

**建议创建的动作**:
| 名称 | 代码 | 说明 |
|------|------|------|
| 攻击 | attack | 普通攻击 |
| 走位 | walk | 移动动作 |
| 技能 | skill | 普通技能 |
| 大招 | ultimate | 终极技能 |
| 回城 | recall | 回城动作 |
| 嘲讽 | joke | 表情动作 |

**注意**:
- ✅ 动作是**全局的**，所有游戏共享
- ✅ 不同游戏可以有相同动作（攻击、走位）
- ❌ 创建动作时**不需要选择角色**（动作与角色通过上传视频时绑定）

---

### 2️⃣ 创建分类（用于角色分类）

**访问**: http://localhost:3003/categories

**操作**:
1. 选择游戏：英雄联盟
2. 点击 **"➕ 新建分类"**
3. 填写信息：
   - **名称**: 战士
   - **层级**: 1 级（顶级分类）
   - **父分类**: 无（1 级分类没有父级）
   - **排序**: 0
   - **图标**: （可选）
4. 点击 **"保存"**

**示例分类结构**:

#### 英雄联盟
```
英雄联盟
├── 战士（1 级）
├── 法师（1 级）
├── 刺客（1 级）
├── 坦克（1 级）
├── 射手（1 级）
└── 辅助（1 级）
```

#### 原神
```
原神
├── 战斗（1 级）
│   ├── 火元素（2 级）
│   ├── 冰元素（2 级）
│   ├── 风元素（2 级）
│   ├── 雷元素（2 级）
│   └── 岩元素（2 级）
└── 表演（1 级）
    └── 角色演示（2 级）
```

**注意**:
- ✅ 分类是**按游戏组织**的，不同游戏有独立分类
- ✅ 支持最多 7 级分类
- ✅ 分类用于**角色分类**和**视频筛选**

### 3️⃣ 创建角色

**访问**: http://localhost:3003/characters

**操作**:
1. 点击 **"➕ 新建角色"**
2. 填写信息：
   - **名称**: 盖伦
   - **游戏**: 英雄联盟
   - **分类**: 战士（从下拉框选择）
   - **头像**: （可选，上传角色头像）
   - **描述**: 德玛西亚之力，盖伦（可选）
   - **排序**: 0
   - **发布**: ✅ 勾选
3. 点击 **"保存"**

**示例角色**:

#### 英雄联盟
| 名称 | 分类 | 说明 |
|------|------|------|
| 盖伦 | 战士 | 德玛西亚之力 |
| 安妮 | 法师 | 黑暗之女 |
| 泰隆 | 刺客 | 刀锋之影 |
| 蕾欧娜 | 坦克 | 曙光女神 |
| 艾希 | 射手 | 寒冰射手 |
| 索拉卡 | 辅助 | 众星之子 |

**注意**:
- ✅ 角色**必须属于某个游戏**
- ✅ 角色**选择分类**（从已有分类中选择）
- ✅ 角色名称在同一游戏内**唯一**

---

### 4️⃣ 上传视频（绑定角色 + 动作）

**访问**: http://localhost:3003/videos

**操作**:
1. 选择游戏：英雄联盟
2. 点击 **"📤 上传视频"**
3. 填写信息：

   **① 选择角色** ⭐ 必选
   ```
   [下拉框：盖伦（战士）]
   ```

   **② 选择动作** ⭐ 必选
   ```
   [下拉框：攻击 (attack)]
   ```

   **③ 选择分类**（可选，用于前台筛选）
   ```
   [按钮：战士] [按钮：法师] [按钮：刺客] ...
   ```

   **④ 选择标签**（可选，用于搜索）
   ```
   [按钮：连招] [按钮：教学] [按钮：高光] ...
   ```

   **⑤ 选择文件**
   ```
   📤 选择视频文件
   garen-attack.mp4
   ```

4. 点击 **"📤 开始上传"**

**上传流程**:
```
1. 获取七牛云上传凭证
   ↓
2. 上传视频到七牛云
   ↓
3. 自动生成封面图（第 1 秒）
   ↓
4. 创建视频数据库记录
   ↓
5. 创建 CharacterAction 关联（角色 + 动作 + 视频）
   ↓
6. 完成！
```

**验证**:
- 访问前台：http://localhost:5173/games
- 选择游戏：英雄联盟
- 选择角色：盖伦
- 应该看到动作按钮：**攻击**
- 点击"攻击"按钮，播放上传的视频

## 常见问题

### Q1: 创建动作时为什么没有选择角色？

**A**: 动作是**全局的**，不直接绑定角色。

**设计逻辑**:
```
动作 (Action) - 全局定义
  ├── 攻击 (attack) - 所有角色共享
  ├── 走位 (walk) - 所有角色共享
  └── 技能 (skill) - 所有角色共享

角色 (Character) - 独立存在
  ├── 盖伦
  └── 安妮

角色动作关联 (CharacterAction) - 绑定关系
  ├── 盖伦 + 攻击 = 盖伦的攻击视频
  └── 安妮 + 攻击 = 安妮的攻击视频
```

**好处**:
- ✅ 动作复用 - 不需要为每个角色重复创建"攻击"动作
- ✅ 灵活性 - 盖伦可以有 4 个动作，安妮可以有 6 个动作
- ✅ 数据规范 - 动作名称和代码统一管理

---

### Q2: 上传视频时找不到角色？

**A**: 需要先创建角色。

**解决步骤**:
1. 访问 **角色管理**: http://localhost:3003/characters
2. 点击 **"➕ 新建角色"**
3. 填写信息（名称、游戏、分类）
4. 保存后，重新打开上传弹窗，角色选择器中就会显示

---

### Q3: 上传视频时找不到动作？

**A**: 需要先创建动作。

**解决步骤**:
1. 访问 **动作管理**: http://localhost:3003/actions
2. 点击 **"➕ 新建动作"**
3. 填写信息（名称、代码）
4. 保存后，重新打开上传弹窗，动作选择器中就会显示

---

### Q4: 一个角色可以有多个动作吗？

**A**: 可以！

**示例**:
```
盖伦
├── 攻击 (attack) - 上传盖伦攻击视频
├── 走位 (walk) - 上传盖伦走位视频
├── 技能 (skill) - 上传盖伦 QWER 技能视频
└── 大招 (ultimate) - 上传盖伦大招视频
```

**操作**:
- 上传 4 次视频，每次选择不同的动作
- 系统会自动创建 4 个 CharacterAction 关联

---

### Q5: 多个角色可以有相同的动作吗？

**A**: 可以！

**示例**:
```
攻击 (attack)
├── 盖伦的攻击视频
├── 安妮的攻击视频
├── 泰隆的攻击视频
└── ...
```

**操作**:
- 上传视频时，选择不同角色 + 相同动作
- 系统会自动创建不同的 CharacterAction 关联

---

### Q6: 上传视频后前台不显示？

**A**: 检查以下几点：

1. **角色是否发布**
   - 访问角色管理
   - 确保角色的"发布"状态是 ✅

2. **动作是否发布**
   - 访问动作管理
   - 确保动作的"发布"状态是 ✅

3. **视频是否发布**
   - 访问视频管理
   - 确保视频的"发布"状态是 ✅

4. **CharacterAction 关联是否创建**
   - 上传视频时必须选择角色和动作
   - 检查后端日志：`grep CharacterAction /tmp/server.log`

5. **前台缓存**
   - 硬刷新浏览器：Ctrl+Shift+R

---

## 数据关系图

```
游戏 (Game)
├── 分类 (GameCategory)
│   └── 角色 (Character)
│       └── 动作关联 (CharacterAction)
│           └── 视频 (Video)
├── 视频 (Video) [直接关联]
└── 动作 (Action) [全局共享]
```

**关键点**:
- ✅ 动作是全局的，不属于某个游戏
- ✅ 角色属于某个游戏
- ✅ 分类属于某个游戏
- ✅ 视频属于某个游戏
- ✅ CharacterAction 关联角色、动作、视频
