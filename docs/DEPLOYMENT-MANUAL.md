## 1. 准备工作

### 1.1 你需要准备的东西

| 物品 | 用途 | 是否必需 |
|------|------|----------|
| 阿里云账号 | 登录服务器 | ✅ 必需 |

**步骤 5**：记住储存空间名称（如 `ckanim-video`）

## 2. 连接服务器

### 2.1 使用阿里云 Workbench（推荐）⭐

**如果你通过阿里云 Workbench 连接**：

**步骤 1**：登录阿里云控制台 https://console.aliyun.com

**步骤 2**：点击左侧 **云服务器 ECS**

**步骤 3**：找到你的服务器（IP: 39.102.115.79）

**步骤 4**：点击 **远程连接** 或 **Workbench**

**步骤 5**：输入服务器密码

**步骤 6**：点击 **登录**

**成功的标志**：看到类似 `[root@iZxxx ~]#` 的提示符

**✅ 优点**：
- 不需要安装任何软件
- 直接在浏览器里操作
- 支持复制粘贴

### 2.2 Windows 用户（cmd/PowerShell）

**步骤 1**：按键盘 `Win + R` 键（Win 键是微软标志那个键）

**步骤 2**：在弹出的框里输入 `cmd`

**步骤 3**：按回车，会出现一个黑色窗口

**步骤 4**：在黑色窗口里输入以下命令（可以复制粘贴）：

```bash
ssh root@39.102.115.79
```

**步骤 5**：按回车

**步骤 6**：如果提示 `Are you sure you want to continue connecting?`

输入 `yes` 并按回车

**步骤 7**：提示 `root@39.102.115.79's password:`

输入你的服务器密码

**⚠️ 注意**：输入密码时**屏幕上不会显示任何字符**，这是正常的！

**步骤 8**：按回车

**步骤 9**：如果看到类似 `[root@iZxxx ~]#` 的提示符，说明连接成功！

### 2.3 Mac 用户

**步骤 1**：按键盘 `Cmd + Space` 键

**步骤 2**：输入 `Terminal`

**步骤 3**：按回车，会出现一个白色窗口

**步骤 4**：在白色窗口里输入：

```bash
ssh root@39.102.115.79
```

**步骤 5**：按回车，后续步骤同 Windows

### 2.4 如果连接失败

**错误 1**：`Connection refused`

**解决**：
1. 打开浏览器：https://console.aliyun.com
2. 登录阿里云
3. 点击 **云服务器 ECS**
4. 点击你的服务器
5. 点击 **安全组**
6. 点击 **手动添加**
7. 端口范围：`22/22`
8. 授权对象：`0.0.0.0/0`
9. 点击 **保存**

**错误 2**：`Permission denied, please try again`

**解决**：密码错了，重新输入。如果忘记密码：
1. 登录阿里云控制台
2. 云服务器 ECS → 实例
3. 找到你的服务器
4. 点击 **更多** → **密码/密钥** → **重置实例密码**
5. 设置新密码
6. 重启服务器

## 3. 安装软件

**💡 提示**：以下每个命令都可以**复制粘贴**！

**如何复制粘贴**：
- **Workbench 用户**：直接复制命令，在终端里右键粘贴（或 Ctrl+V）
- **Windows 用户**：复制命令，在 cmd 窗口里右键点击（自动粘贴）
- **Mac 用户**：复制命令，在 Terminal 里按 `Cmd + V`

**⚠️ 注意**：每个命令都要**单独输入**，输入后按回车，等待完成后再输入下一个。

### 3.1 更新系统

输入：

```bash
yum update -y
```

**说明**：更新系统软件，需要 2-5 分钟。

**成功的标志**：最后出现 `Complete!`

### 3.2 安装 Node.js

输入：

```bash
curl -fsSL https://rpm.nodesource.com/setup_20.x | bash -
```

**说明**：下载 Node.js 安装脚本，会看到很多滚动文字。

**成功的标志**：最后出现 `done` 或 `Complete`

然后输入：

```bash
yum install -y nodejs
```

**说明**：安装 Node.js，需要 1-3 分钟。

**成功的标志**：出现 `Complete!`

验证安装：

```bash
node -v
```

**成功的标志**：显示 `v20.x.x`（x 是数字）

### 3.3 安装 Git

输入：

```bash
yum install -y git
```

**说明**：安装 Git（用于下载代码）。

**成功的标志**：出现 `Complete!`

验证安装：

```bash
git --version
```

**成功的标志**：显示 `git version 2.x.x`

### 3.4 安装 SQLite

输入：

```bash
yum install -y sqlite
```

**说明**：安装数据库。

**成功的标志**：出现 `Complete!`

验证安装：

```bash
sqlite3 --version
```

**成功的标志**：显示版本号如 `3.34.1`

### 3.5 安装 PM2

输入：

```bash
npm install -g pm2
```

**说明**：安装 PM2（用于让网站持续运行）。

**成功的标志**：出现 `added x packages`

验证安装：

```bash
pm2 -v
```

**成功的标志**：显示数字如 `5.3.0`

## 5. 配置七牛云

### 5.1 生成随机密钥（JWT_SECRET）

**先执行这个命令**（生成随机字符串）：

```bash
openssl rand -base64 32
```

**说明**：生成一个随机字符串作为 JWT 密钥。

**成功的标志**：显示一串类似 `xK9mL2nP5qR8sT1vW4yZ7aB0cD3eF6gH9iJ2kL5mN8o= `的字符串

**⚠️ 重要**：复制这个字符串，后面要用！

### 5.2 创建配置文件（使用 nano 编辑器 - 简单！）⭐

**输入**：

```bash
nano .env.production
```

**说明**：使用 nano 编辑器创建配置文件（比 vi 简单得多！）

**成功的标志**：打开一个空白编辑界面

### 5.3 输入配置内容

**现在可以输入文字了**，把以下内容复制粘贴进去：

```
# JWT 配置（⚠️ 生产环境必须修改！）
JWT_SECRET="粘贴你刚才生成的随机字符串"
JWT_EXPIRES_IN="7d"

# 七牛云配置（⚠️ 必须修改成你的！）
# 获取方式：七牛云控制台 → 对象存储 → 密钥管理/储存空间
QINIU_ACCESS_KEY="你的 AccessKey"
QINIU_SECRET_KEY="你的 SecretKey"
QINIU_BUCKET="你的储存空间名称（如 zhuque-guangdong）"
QINIU_DOMAIN="https://你的七牛云域名（如 https://video.jiangmeijixie.com）"
QINIU_PREFIX="你的路径前缀（如 参考网站 2026/）"

# 数据库配置（SQLite）
# 说明：使用绝对路径，避免在不同目录执行时出错
# 实际文件位置：/var/www/ckanim/server/prisma/dev.db
DATABASE_URL="file:/var/www/ckanim/server/prisma/dev.db"

# 服务器配置
PORT=3002
NODE_ENV=production
```

# 前台配置
FRONTEND_PORT=5173
FRONTEND_HOST=0.0.0.0

# 后台配置
ADMIN_PORT=3003
ADMIN_HOST=0.0.0.0

# 后端配置
SERVER_PORT=3002
SERVER_HOST=0.0.0.0
```

### 5.4 修改七牛云配置

**⚠️ 必须修改以下内容**：

```
JWT_SECRET="粘贴你刚才生成的随机字符串"     ← 粘贴第 5.1 步生成的字符串
QINIU_ACCESS_KEY="你的 AccessKey"          ← 你的七牛云 AccessKey
QINIU_SECRET_KEY="你的 SecretKey"          ← 你的七牛云 SecretKey
QINIU_BUCKET="你的储存空间名称"            ← 你的七牛云储存空间名称（不是域名！）
QINIU_DOMAIN="https://你的七牛云域名"      ← 你的七牛云域名（要带 https://）
QINIU_PREFIX="你的路径前缀"                ← 你的文件路径前缀（可选）
```

**示例**（假设你的配置是）：
```
JWT_SECRET="xK9mL2nP5qR8sT1vW4yZ7aB0cD3eF6gH9iJ2kL5mN8o="
QINIU_ACCESS_KEY="DwLK5ft-Zx0XgxiI8HaIyeUh0wyaHddssczs2s0c"
QINIU_SECRET_KEY="14ykOp2Q-nkbLmSfZdd2aHmoEnZUHqWxk1BeFN2-"
QINIU_BUCKET="zhuque-guangdong"
QINIU_DOMAIN="https://video.jiangmeijixie.com"
QINIU_PREFIX="参考网站 2026/"
```

**如何编辑**：
- 使用方向键移动光标
- 删除原有文字
- 输入或粘贴你的配置

**⚠️ 重要提示**：
- `QINIU_BUCKET` 填储存空间名称（如 `zhuque-guangdong`），不是域名！
- `QINIU_DOMAIN` 要带 `https://`（如 `https://video.jiangmeijixie.com`）
- `QINIU_PREFIX` 用于组织文件，可以是中文（如 `参考网站 2026/`）

### 5.5 保存并退出（nano 编辑器）

**步骤 1**：按 `Ctrl + O` 键（保存）

**步骤 2**：按回车（确认文件名）

**步骤 3**：按 `Ctrl + X` 键（退出）

**说明**：nano 比 vi 简单多了！

**成功的标志**：回到命令行提示符

### 5.6 如果不小心搞乱了

**按 `Ctrl + X`**，然后按 `N`（不保存退出）

然后重新执行 5.2-5.5 步骤。

### 5.7 验证配置文件

输入：

```bash
cat .env.production
```

**说明**：查看配置文件内容。

**成功的标志**：显示你刚才输入的配置内容

## 12. 常见问题

### Q1: 连接服务器时提示"连接被拒绝"

**A**: 阿里云安全组没有开放 22 端口

**解决**：
1. 登录阿里云控制台
2. 云服务器 ECS → 安全组
3. 添加入方向规则：端口 22，授权对象 0.0.0.0/0

### Q2: 命令执行失败，提示"command not found"

**A**: 软件没有安装成功

**解决**：重新执行安装命令，确保看到 `Complete!` 或 `added x packages`

### Q3: 网站访问很慢

**A**: 服务器带宽只有 3Mbps，属于正常现象

**解决**：
- 视频建议使用七牛云存储（不占用服务器带宽）
- 考虑升级服务器带宽

### Q4: 服务运行一段时间后停止

**A**: PM2 没有配置开机自启

**解决**：
```bash
pm2 startup
pm2 save
```

### Q5: 数据库出错

**A**: 数据库文件损坏或权限问题

**解决**：
```bash
cd /var/www/ckanim/server
npx prisma migrate reset
```

**⚠️ 警告**：这会清空所有数据！

### Q6: 七牛云视频无法播放

**解决**：
1. 检查七牛云配置是否正确
2. 检查储存空间是否是公开读
3. 检查域名是否正确

### Q7: 上传视频失败

**解决**：
1. 检查七牛云 AccessKey 和 SecretKey 是否正确
2. 检查储存空间名称是否正确
3. 检查服务器时间是否准确
