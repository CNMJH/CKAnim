## 📋 管理操作指南

### 初始化默认套餐
1. 登录管理后台
2. 访问 VIP 套餐管理页面
3. 点击"初始化默认套餐"按钮
4. 确认后即可创建 3 个默认套餐

### 添加新套餐
1. 点击"+ 添加套餐"按钮
2. 填写套餐信息（名称、等级、价格、权益等）
3. 点击"创建套餐"

### 编辑套餐
1. 点击套餐行的"编辑"按钮
2. 修改套餐信息
3. 点击"保存更新"

### 删除套餐
1. 点击套餐行的"删除"按钮
2. 确认删除
