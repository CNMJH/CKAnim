## 📝 清理原因

这些孤文件产生的原因：
1. **测试数据** - 开发过程中上传的测试视频
2. **直接删除数据库** - 没有通过 API 删除，导致七牛云文件残留
3. **图标替换** - 上传新图标后旧文件未清理

## 🎯 预防措施

### 正确删除流程

✅ **推荐**: 使用管理员后台删除
```
后台点击删除
    ↓
API 调用
    ↓
删除数据库记录
    ↓
删除七牛云文件
    ↓
完全清理 ✅
```

❌ **避免**: 直接操作数据库或七牛云
```
直接删除数据库记录
    ↓
七牛云文件残留 ❌
```

### 定期维护

建议每周运行一次同步检查：
```bash
# 添加到 crontab，每周日凌晨 3 点
0 3 * * 0 cd /home/tenbox/CKAnim/server && node scripts/sync-check.js >> /var/log/ckanim-sync.log 2>&1
```
